# Laboratorio de Double Negatives Web

Aplicación web didáctica para practicar la corrección de **double negatives** en inglés.

El proyecto transforma una actividad de gramática sobre cómo evitar dobles negaciones en una herramienta interactiva con explicación, práctica, retroalimentación inmediata, quiz final y registro de progreso en el navegador.

---

## Objetivo del proyecto

Crear una herramienta de práctica para identificar y corregir oraciones con double negatives.

El sistema permite:

- Consultar una explicación breve del tema.
- Revisar negative words.
- Revisar positive replacements.
- Elegir la palabra correcta para completar oraciones.
- Reescribir oraciones con doble negación.
- Ver dos formas correctas de una misma oración.
- Resolver un quiz final.
- Guardar progreso local en el navegador.

## Módulos del proyecto

### Explicación

Presenta qué es un double negative y cómo corregirlo.

Ejemplo:

```text
Double negative:
I don’t need no help.

Correction 1:
I don’t need any help.

Correction 2:
I need no help.
```

---

### Elegir palabra

El usuario selecciona la palabra correcta para evitar doble negación.

Ejemplo:

```text
I can’t find my notes ___.
nowhere / anywhere
```

---

### Reescribir

El usuario escribe dos versiones correctas de una oración con double negative.

Ejemplo:

```text
Original:
I didn’t ask nobody for help.

Correction 1:
I didn’t ask anybody for help.

Correction 2:
I asked nobody for help.
```

---

### Quiz final

El usuario responde preguntas de opción múltiple y recibe:

- Aciertos
- Porcentaje
- Retroalimentación por pregunta
- Mejor puntaje guardado

## Enfoque académico

Este laboratorio convierte un ejercicio de gramática en inglés en una aplicación de práctica interactiva.

El objetivo es reforzar el uso correcto de una sola negación en oraciones negativas y distinguir entre palabras negativas y reemplazos positivos como `any`, `anybody`, `anything` y `anywhere`.
